# run the demo
```
cd example #enter example forder
npm install
npm run serve
```


# install vue.je (application from scratch)
```
npm_config_unsafe_perm=true # if you are root
npm install -g @vue/cli
vue create ui
npm install vue-router@4
npm install axios
```
